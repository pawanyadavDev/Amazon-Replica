<h2>Amazon-Replica</h2>
The Amazon Replica is a front-end web project developed using HTML and CSS, designed to replicate the visual layout and user interface of the Amazon e-commerce website. This project focuses on creating a clean, responsive, and user-friendly design that closely resembles the original platform.The replica includes key UI components such as the header navigation bar, search section, product listings, category sections, banners, and footer. Emphasis was placed on proper page structure, consistent styling, alignment, and layout techniques using modern CSS concepts such as Flexbox and Grid.

This project helped strengthen foundational skills in web development, UI/UX design, and responsive layout creation, while enhancing understanding of how large-scale e-commerce websites are structured at the front-end level.
Technologies Used:  HTML5 , CSS3.
